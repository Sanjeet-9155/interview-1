
JSX
jsx stands for js xml or extension.
with the help of jsx 
we can write HTML and Js code in React.
Jsx converts html tags into React elements.

or ham react appliaction me aadha se jadaHTml code. 
likh sakte hai.
reactjs-- App.js me HTml likte hai usko bolte hai JSX.

Rules of writing JXS
-HTML code must wrap into one top element.
-Elemnts must be closed.
-Attribute class = className.
-No if else condition inside but ternary operator is okay.
Js expression in jsx must be wrapped in {}
